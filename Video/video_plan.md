# Dashboard Video Walkthrough Plan

## Format
- 60-90 second screen recording of the live dashboard
- Voiceover narrating (your voice, not scripted word-for-word)
- No face cam, no slides, no transitions, no music
- Mouse cursor guides the viewer's eye (hover, don't click frantically)
- Only actual clicks are the 4 page navigation tabs

## Visual Flow

**0-5s** — P1 (Verdict) loads. KPI cards, S-curve, contradictions table visible.
Voice: One line about what this is and the problem it solves.

**5-20s** — Stay on P1. Hover over contradictions table, then actions log.
Voice: Cross-system contradiction detection — what it catches, why it matters.

**20-35s** — Click to P2 (Early Warning). Float erosion chart, then scroll to production validation.
Voice: Problems show up here before they hit reports. Float burning, production rate gaps.

**35-50s** — Click to P3 (Recovery Reality). Recovery validation cards, subcontractor EVM table.
Voice: Contractor says they'll recover — historical data says otherwise.

**50-58s** — Click to P4 (Control Integrity). Source systems staleness cards, completeness matrix.
Voice: None of this matters if the data is stale. This page checks the data itself.

**58-60s** — Pause. Clean ending.
Voice: One closing line — what this demonstrates, where to reach you.

## Script Style
- NOT word-for-word reading — bullet points on a sticky note
- One key idea per page, say it in your own words
- If you stumble, pause 2 seconds, repeat the line, trim later

## Recording Setup
- Record each page separately (5 clips of ~15s) OR one continuous take
- Trim bad takes in Clipchamp (built into Windows 11)
- If recording as separate clips, stitch in Clipchamp

## Cursor
- Option A: Microsoft PowerToys → Mouse Utilities → highlight circle around cursor (easiest)
- Option B: JavaScript cursor follower built into the dashboard (cleaner on video, needs removal after)

## Audio
- External mic or headset → record as-is, no filtering needed
- Laptop built-in mic → use Audacity noise reduction (record 2s silence → get profile → apply)
- Worst case → record voice on phone, drop onto screen recording in Clipchamp

## Deliverable
- ~60-90 second MP4
- Can be shared as a link alongside the PDF and dashboard URL
